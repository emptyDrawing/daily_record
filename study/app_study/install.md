### install 

1. node.js + watchman(build tool)

```bash
brew install node
brew install watchman
```

2. react-navtive install

```bash
npm install -g react-native-cli
npx react-native --version
```

3. Cocoapods 설치

```bash
brew install cocoapods
pod --version
```

4. init project

```bash
npx react-native init {project-name}
```
